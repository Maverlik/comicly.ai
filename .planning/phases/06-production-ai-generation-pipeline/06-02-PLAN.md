<threat_model>
- **Threat:** OpenRouter API key leaks through logs/errors/responses. **Mitigation:** Redact provider errors to stable codes/messages and never include secrets in payloads.
- **Threat:** Malformed provider responses create successful pages without images. **Mitigation:** Treat missing image as provider failure and cover with fixture tests.
- **Threat:** Large base64 reaches frontend response. **Mitigation:** Storage service converts provider image output to Blob URL and downstream APIs return only URL/storage key.
</threat_model>
