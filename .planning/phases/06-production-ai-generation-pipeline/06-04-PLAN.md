<threat_model>
- **Threat:** Anonymous users can spend provider credits or infer private data. **Mitigation:** Use `get_current_user` for both generation and AI text routes.
- **Threat:** Client sends owner/balance/cost fields. **Mitigation:** Do not accept trusted owner, balance, or coin cost from request; service derives these server-side.
- **Threat:** API response accidentally includes provider base64 or secret-bearing payloads. **Mitigation:** Response models expose only job/page/balance and durable image URL/storage key.
</threat_model>
