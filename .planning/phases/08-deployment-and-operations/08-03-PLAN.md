<threat_model>
- **T-08-09 Threat:** Operators miss a required env var or use pooled DB URLs for migrations incorrectly. **Mitigation:** Env docs distinguish runtime pooled `DATABASE_URL` from direct migration URL.
- **T-08-10 Threat:** Secrets are copied into docs/examples. **Mitigation:** Examples use placeholders only and tests check for secret-like committed values where practical.
- **T-08-11 Threat:** OAuth fails in production due to wrong callback/domain/cookie settings. **Mitigation:** Deployment runbook lists exact Google/Yandex callback URLs, cookie domain, secure cookie, and CORS settings.
- **T-08-12 Threat:** Smoke checks imply provider flows passed when they were not run. **Mitigation:** Smoke helper/runbook separates automated checks from manual live OAuth/generation checks.
</threat_model>
