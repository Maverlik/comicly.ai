<threat_model>
- **Threat:** Frontend work starts from a stale branch and overwrites current main changes. **Mitigation:** First task fetches and merges/rebases `origin/main`, resolving conflicts before frontend edits.
- **Threat:** Landing page accidentally requires auth. **Mitigation:** Auth gate is scoped only to `create.html` and verified by static/browser checks.
- **Threat:** Browser fetches miss auth cookies cross-origin. **Mitigation:** Central API helper uses `credentials: "include"` for backend API calls.
- **Threat:** Fake profile/credits remain trusted in production. **Mitigation:** Session bootstrap owns profile/balance state before creator is interactive.
</threat_model>
